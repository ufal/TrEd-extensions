=head1 NAME

PhraseTrees - Context for Annotation of Constituency Syntax in the TrEd Environment


=head1 REVISION

    $Revision: 542 $       $Date: 2008-04-25 23:34:45 +0200 (Fri, 25 Apr 2008) $


=head1 DESCRIPTION

For reference, see the list of PhraseTrees macros and key-bindings in the User-defined menu item in TrEd.




=head1 BINDINGS

=over

=item 'BackSpace'

Annotate: Undo recent annotation action C<invoke_undo>

=item 'CTRL+End'

Move to Leftmost Descendant C<move_deep_end>

=item 'CTRL+F1'

Show / Hide Morphological Tags 1 C<toggle_tag_1>

=item 'CTRL+F2'

Show / Hide Morphological Tags 2 C<toggle_tag_2>

=item 'CTRL+F3'

Show / Hide Morphological Tags 3 C<toggle_tag_3>

=item 'CTRL+Home'

Move to Rightmost Descendant C<move_deep_home>

=item 'CTRL+Return'

ThisAddress() to Clipboard C<ThisAddressClipBoard>

=item 'CTRL+SHIFT+Down'

Move Down to Fork C<move_to_fork>

=item 'CTRL+SHIFT+Up'

Move Up to Root C<move_to_root>

=item 'CTRL+equal'

Toggle Children Hiding C<tree_hide_mode>

=item 'CTRL+j'

Toggle Tree Justify Mode C<tree_justify_mode>

=item 'CTRL+l'

Display Trees Left-to-Right C<direction_LTR>

=item 'CTRL+minus'

Hide Children Subtrees C<hide_children>

=item 'CTRL+plus'

Unhide Children Recursively C<unhide_subtree>

=item 'CTRL+r'

Display Trees Right-to-Left C<direction_RTL>

=item 'CTRL+underscore'

Hide This Subtree C<hide_this>

=item 'End'

Move to Last Word C<move_word_end>

=item 'Home'

Move to First Word C<move_word_home>

=item 'SHIFT+End'

Move to Last Paragraph C<move_par_end>

=item 'SHIFT+Home'

Move to First Paragraph C<move_par_home>

=item 'SHIFT+Next'

Move to Next Paragraph C<move_to_next_paragraph>

=item 'SHIFT+Prior'

Move to Prev Paragraph C<move_to_prev_paragraph>

=item 'exclam'

Annotate: Edit annotator's comment C<edit_comment>

=back
