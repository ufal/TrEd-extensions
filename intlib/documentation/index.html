TrEd Extension: intlib (version 1.0)

Czech Legal Text Annotation

Description:

Provides annotation mode for the Czech Legal Text Treebank (CLTT) project.

Installation

This extension can be installed directly from TrEd, using Setup >> Manage Extensions >> Get New Extensions; just make sure the repository http://ufal.mff.cuni.cz/tred/extensions/core/ is enabled in Setup >> Manage Extensions >> Edit Repositories.

TrEd macro files:

  • intlib/PML_INTLIB_A_Edit.mak

Resource files:

  • pml_schema_w.xml
  • pml_schema_m.xml
  • pml_schema_a.xml
  • pml_schema_a_intlib.xml
