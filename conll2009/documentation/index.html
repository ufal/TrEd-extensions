CoNLL 2009 Shared Task Data Visualization

This package contains scripts and a stylesheet used for creating the visualization of the CoNLL-2009-ST trial data. It can be used to convert any data from CoNLL-2009-ST format to PML so that they can be opened by TrEd.

NOTE: Works only on Linux and similar systems. XSH and bash must be installed.

Usage

Run the bash script 

conll2009/bin/conll2009-to-pml.sh

with CoNLL-2009-ST data files as arguments. When finished, open the created *.pml data in tred, using the stylesheet conll2009.
